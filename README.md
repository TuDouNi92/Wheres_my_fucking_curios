# Where is my fxxking Curios
## This plugin requires hybrid Minecraft server to run (Technically the environment which supports Spigot & Forge)
### Intro
This plugin remedy the desync on curios items problem when using multiverse-inventories.
### Tested version/requiring Dependencies:
- Minecraft Forge 1.20.1-47.2.0
  - CuriosApi 1.20.1-5.3.1
  - Sophisticated core 1.20.1-1.3.1
  - Sophisticated backpacks 1.20.1-3.24.20.1500
- Mohist 1.20.1 2025Q4 vers
  - Multiverse-core 5.5.1
  - Multiverse-inventories 5.3.1
### Usage
  - Install the plugin and make sure you have all the requirements
  - `/mvinv info curios` to check if sharable group `curios` exists, it applies to vanilla worlds by default
  - The share name is `curioSharable` or alias `cs` for editing group shares. For usages, see multiverse-inventories wiki
